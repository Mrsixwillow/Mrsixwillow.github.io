如何搭建自己的博客

下载git和node

进行安装

点击进行git下载git下载

点击进行node下载node下载

如果下载过慢可直接再电脑自带的应用商店中下载或者找镜像网站下载

下载完毕后可以在gitee或者cnsd上找安装git和node的步骤

node安装完成后

（1）Node安装

按照安装过程依次点击即可，安装过程很简单。
（2）安装Git

按照网上的安装方式进行安装就可以了
https://www.cnblogs.com/ximiaomiao/p/7140456.html
（3）安装Hexo

（下面出现的检测全部是在git中进行的即在git黑框中敲入这些字母）

检测node安装是否成功：

键入node -v

（返回版本号）
键入npm -v

（返回版本号）

安装淘宝的镜像源（如果错误就重来，因为连接可能不稳定，不要害怕多试几次就可以了）

键入npm install -g cnpm –registry=https://registry.npm.taobao.org
检测

键入cnpm

（返回cnpm的信息）

键入cnpm -v

（返回cnpm的版本信息）

安装hexo（如果失败重新来过就可以了）

键入cnpm install -g hexo-cli

检测

键入hexo -v

（返回hexo的版本信息）
